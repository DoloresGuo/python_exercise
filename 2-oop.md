# 3.面向对象的三大特性
## 3.1 封装
- 对对象的成员进行访问限制
- 封装的三个级别：
    - 公开：public
    - 受保护的：protected
    - 私有的：private
- 判别对象的位置
    - 对象内部
    - 对象外部
    - 子类中
    
- 私有
    - 最高级别的封装
    - 在成员前面加两个下划线即可
    
        class Person():
            # name是公有成员
            name = "liuyan"
            # __age就是私有成员
            __age = 19
- 受保护的封装
    - 只有类中的子类可以进行访问
    - 在成员前加一个下划线
- 公开的
    - 公共的封装对成员没有任何限制，内外成员都可以访问
## 3.2 继承
- 继承就是一个类可以获得另外一个类中的成员属性和成员方法
